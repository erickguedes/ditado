# Contributing

Thanks for considering contributing to Ditado!

## Development Setup

```bash
git clone <your-fork>
cd ditado
pip install faster-whisper sounddevice numpy pyperclip pystray pillow
```

## Code Style

- Follow PEP 8
- Keep it simple — this is a single-file app
- Docstrings in English for all public functions

## Pull Request Process

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/my-change`)
3. Make your changes
4. Verify the app still runs: `python ditado.py`
5. Open a PR describing what you changed and why

## Reporting Issues

Open a GitHub issue with:
- Steps to reproduce
- Windows version
- Python version
- Relevant logs or error messages
